# Wellesley Rate My Professor

 Wellesley rate my professor is a web app that allows student at Wellesley college to find which professors are best for them based on their learning style and the grade they want to acheive. 

## Contents

* `site.html` This file creates the main page of the website and loads all of the CS professor profile cards.
* `profrate.mjs` This file creates faculty cards and renders these cards in a grid format, it also controls the search function where users are able to search for a specific professor they want to see. 
* `viewProfProfile.html` This file creates the page you are sent to after clicking on a specific professor to see their individual stats as well as the data associated with your learning style and desired grades. 
* `viewProfProfile.mjs` This file ensures that when clicking on an individual professor, you are sent to a profile page that that their specific info and not, for example, a different professor or random base set of content. 
* `data.js` This file contains all of the data for the CS professors which was scraped from the Wellesley Website with their names, headshots, and roles. 
* `styles.css` This file contains all of the styling for all of the html files.
* `scraping.py ` This file scrapes info from the Wellesley Website that contains profiles for all the Wellesley CS professors. 
* `studentdata.py` This file contains fake date aboout students and their grade preferances and learnign styles.
* `Carolyn Anderson_grades.csv` This is a csv file that contains fake data about the kinds of grades that Carolyn Anderson gives
* `Eni Mustafaraj_grades.csv` This is a csv file that contains fake data about the kinds of grades that Eni Mustafaraj gives
* `Panagiotis Metaxas_grades.csv` This is a csv file that contains fake data about the kinds of grades that Panaiotis Metaxas gives
* `visualisation.py` 
* `about.html` This is an about page for the website -- it does not contain much. 
* `account.html` This is an account page for the website that does absolutely nothing.
